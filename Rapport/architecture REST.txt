Ressource : Canaux (/api/channels)

Méthode HTTP         Endpoint                             Description                          Code Statut (Succès)          Code Statut (Erreur)
GET                /api/channels   Récupère la liste de tous les canaux publics disponibles.          200 OK          401 Unauthorized (Si non connecté)
POST               /api/channels           Crée un nouveau canal (public ou privé).                 201 Created       400 Bad Request / 401 Unauthorized



Exemple de réponse JSON (GET /api/channels)

JSON
[
  {
    "id": 1,
    "name": "General",
    "type": "public",
    "creatorId": 1
  }
]







Ressource : Messages (/api/messages ou /api/channels/{id}/messages)

Méthode HTTP         Endpoint                                     Description                                 Code Statut (Succès)                 Code Statut (Erreur)
GET           /api/channels/{id}/messages     Récupère l'historique des messages d'un canal spécifique.            200 OK              403 Forbidden (Si canal privé et non membre)
POST               /api/messages                         Publie un message dans un canal.                          201 Created                400 Bad Request / 403 Forbidden
PUT              /api/channels/{id}                 Modifie le contenu d'un message existant.                      200 OK            403 Forbidden (Si l'utilisateur n'est pas l'auteur)
DELETE           /api/messages/{id}                    Supprime un message du système.                           204 No Content        403 Forbidden (Si ni auteur, ni admin du canal)


Exemple de corps de requête JSON (POST /api/messages) :

JSON
{
  "channelId": 1,
  "content": "Salut l'équipe ! Quelqu'un a fini le Jalon 5 ?"
}

Exemple de réponse JSON (GET /api/channels/1/messages) :
JSON
[
  {
    "id": 1,
    "content": "Hello tout le monde !",
    "createdAt": "2026-06-03 12:53:03",
    "userId": 1,
    "channelId": 1
  }
]






Ressource : Membres / Adhésions (/api/members)

Méthode HTTP          Endpoint                              Description                               Code Statut (Succès)                Code Statut (Erreur)
POST                /api/members      Ajoute ou invite un utilisateur dans un canal (Rejoindre).          201 Created          403 Forbidden (Si canal privé sans droit)
GET                 /api/members               Liste les membres ayant accès au canal.                       200 OK                         401 Unauthorized


Exemple pour un accès interdit (403 Forbidden) :
JSON
{
  "status": "error",
  "code": 403,
  "message": "Accès refusé : Seuls les membres d'un canal privé peuvent lire et écrire dans ce canal."
}
